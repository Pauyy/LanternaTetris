# LanternaTetris


## Controls
The Controls are saved in ```tetriminoCoordinater```

|Key|Action|
|---|---|
| → | Move Right|
| ← | Move Left|
| ↑, x | Rotate Right|
| z, y | Rotate Left|
| ↓ | Soft Drop|
| space | Hard Drop|


## Features
- Right-handed Nintendo Rotation System
- Different Tetromino Randomizer
- Game Over Screen
- Hard Drop
- Soft Drops
- Block Out
- 10x22 Board
- DAS with 16 Frames delay
- ARR with 6  Frames delay
- Tetromino Preview
